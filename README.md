# 🚀 SpaceShip
> A basic game made with Java and [LibGDX](https://github.com/libgdx/libgdx).

---

# ⚙️  Dependencies
+ [OpenJDK](https://github.com/openjdk)
+ [Gradle](https://github.com/gradle/gradle)

---

# 🎮 How to play
```bash
git clone https://github.com/thobias90/spaceship
cd spaceship
gradle run
```
